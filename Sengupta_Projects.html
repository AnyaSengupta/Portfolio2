Jurassic Panic

September 2019, Computer Science Principles

The objective of this project was to create a game or story with user interaction and multiple stages in Scratch. My partner and I previously used Scratch, so we decided to challenge ourselves by implementing the video camera feature. The objective of our game was for the user to guide the dinosaur into eating a number of sprites under a time limit. We incorporated the video sensibility by the user moving their body to move the sprites into the mouth of the dinosaur. The game has multiple stages, making the game harder as it goes on by decreasing the time limit and later adding obstacles. As the game progresses, pterodactyls will attempt to stop the user and the game. If the user's body touches the pterodactyl, the game will be over. This was my first group project in high school, as I learned the importance of cooperation and teamwork with my partner in this project. I was in charge of the script and programming while she was in working on the documentation so we needed to cooperate with each other.

Jailbreak

January 2020, Computer Science Principles

The objective of this project was to create an adventure game with a consistent theme and user input that affects the story line in Python. We had to face an obstacle as one of the requirements was no loops which meant we needed to use functions. Our objective was to create a jail-theme where the user must find the key to leave the cell by answering riddles. My partner and I delegated the work by splitting up the branches. I focused on the two of the pathways and my partner did the other two. From the assignments prior to the project, My partner and I developed a very effective and productive relationship.

Santa Catchinator

November - December 2020, Principles Of Engineering

The objective of this project was to design, model, and test an interlocking compound machine that completes the task of capturing “Santa” successfully. The main obstacle we faced in this project was finding materials to use because we could only integrate household items. Each member in my group had to create a part of the Rube Goldberg machine incorporating at least three simple machines like levers, screws, wheel and axle, and pulleys. To demonstrate how our sections would work together, we created a two-dimensional simulation. My group met the requirements and exceeded the expectations even though we faced multiple obstacles.
